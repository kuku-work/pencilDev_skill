# Pencil.dev Style Keywords Reference

## Layout Styles

### Swiss Layout
- Grid-based, mathematical precision
- Strong typography hierarchy
- Asymmetric but balanced
- Clean lines, minimal decoration
- **Use for**: Corporate, editorial, modern brands

### One Column Layout
- Single content flow
- Easy to scan
- Mobile-first friendly
- Strong vertical rhythm
- **Use for**: Blogs, articles, landing pages

### Two-Column Grid
- Content + sidebar structure
- Information hierarchy
- Desktop-optimized
- Flexible content areas
- **Use for**: Dashboards, admin panels, documentation

### Card-Based Layout
- Modular content blocks
- Easy to reorganize
- Responsive-friendly
- Clear content boundaries
- **Use for**: Product grids, portfolios, social feeds

### Sidenav Layout
- Persistent navigation
- Deep hierarchy support
- Maximized content area
- Desktop-focused
- **Use for**: Admin panels, complex apps, documentation

- Rounded corners
- Soft shadows
- Generous white space

- Grid patterns

- **Feeling**: Reliable, corporate, trustworthy

## Design Principles

### Focus on Typography
```
- Headline: 48-64px, bold weight
- Subheadline: 24-32px, medium weight
- Body: 16-18px, regular weight
- Caption: 12-14px, light weight
- Limit to 2 font families maximum
```

### Most Emphasis on [Element]
```
- Make target element largest
- Use brightest/boldest color
- Position prominently (top-left or center)
- Reduce visual weight of other elements
- Add whitespace around focus element
```

### Everything Else Secondary
```
- Reduce contrast of non-essential items
- Use smaller sizes
- Muted colors (grays, lighter tones)
- Position in periphery
- Group related secondary items
```

### Generous White Space
```
- Minimum 24px between sections
- 16px between related items
- 8px between tightly coupled elements
- Empty space is not wasted space
- Let content breathe
```

## Platform-Specific Guidelines

### Mobile App
- Touch targets minimum 44x44px
- Bottom navigation preferred
- Thumb-friendly interactions
- Single-column layouts
- Large, readable text (16px+)

### Web App (Desktop)
- Hover states essential
- Keyboard navigation support
- Dense information acceptable
- Multi-column layouts work
- Sidebar navigation common

### Website (Marketing)
- Above-the-fold impact crucial
- Clear CTAs
- Scrolling storytelling
- Hero sections
- Social proof elements

## Common Style Combinations

### SaaS Dashboard
```
Layout: Sidenav + Two-column
Style: Clean and Professional
Principles: Focus on data, Secondary navigation
Colors: Blue primary, Gray neutrals, Green/Red for status
```

### E-commerce
```
Layout: Card-based grid
Style: Clean with product focus
Principles: Product emphasis, Easy scanning
Colors: Minimal (let products shine), Strong CTAs
```

### Creative Portfolio
```
Layout: Asymmetric grid
Style: Bold and unique
Principles: Typography focus, Generous space
Colors: High contrast, Limited palette
```

### Productivity App
```
Layout: One-column or Kanban
Style: Technical + Clean hybrid
Principles: Information density, Clear hierarchy
Colors: Neutral base, Color coding for categories
```
